# React Native Subscription Skills

RevenueCat subscriptions for React Native.

## Installation

```bash
npx skills add /Users/umituz/Desktop/github/umituz/apps/mobile/npm-packages/react-native-subscription/skills/SKILL.md -g
```

## Usage

Say: **"Setup subscriptions in my app"**

## Features

- RevenueCat integration
- Paywall UI components
- Premium gating (usePremium, useFeatureGate)
- Credit/coin system
- Subscription status tracking
- Entitlement management

## Hooks

- `usePremium()` - Check premium status
- `useFeatureGate()` - Gate specific features
- `useCredits()` - Credit balance
- `useDeductCredit()` - Spend credits
